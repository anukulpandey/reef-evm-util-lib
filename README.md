# Reef EVM util-lib

Utility library for Reef EVM. 

## Run Locally

1. Clone Repository

```
git clone https://github.com/anukulpandey/reef-evm-util-lib
```

2. Install dependencies

```
cd reef-evm-util-lib
yarn install
```

3. Build

```
yarn build
```

## Run Tests

All tests are inside `/test/` directory. You can add new tests there using vitest.

Run tests using

```
yarn test
```
